## excalibur design 简介

学习搭建组件库，制作一个简易版的 antd

地址：https://excalibur-desigin.vercel.app

React + TypeScript + Storybook + 单元测试学习

## 项目使用

使用 `yarn create react-app my-app template --typescript` 创建的项目
安装依赖： `yarn`
启动项目：`yarn storybook`
单元测试：`yarn jest`
